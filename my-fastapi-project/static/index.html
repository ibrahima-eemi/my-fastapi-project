Welcome to My FastAPI Project. This API provides functionalities for managing students and their grades.

API Endpoints

  • POST /token - Obtain a JWT token
  • POST /student - Create a new student
  • GET /student/{student_id} - Get details of a student by ID
  • DELETE /student/{student_id} - Delete a student by ID
  • GET /student/{student_id}/grades/{grade_id} - Get a specific grade for a student
  • DELETE /student/{student_id}/grades/{grade_id} - Delete a specific grade for a student
  • GET /export - Export student data in JSON or CSV format
